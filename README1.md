Базовый вызов функции Реализована простая функция sayHello(), которая выводит сообщение "Hello, JS!". Демонстрируется механизм объявления и вызова функций.
Параметры функции Функция greet(name) принимает имя и выводит соответствующее приветствие. Используется передача аргументов в функцию.
Приветствие с проверкой Функция greet(name) выводит приветствие с именем, если оно указано, либо выводит "Hello, stranger", если имя не передано. Иллюстрируется проверка существования аргумента.
Возведение в квадрат Функция square(x) вычисляет квадрат числа, передаваемого ей в качестве аргумента. Отрабатывается понятие возврата значений функциями.
Сумма массива Функция sumArray(arr) суммирует все элементы числового массива. Применяется цикл for и метод reduce() для демонстрации обработки массивов.
Минимум из двух Функция min(a, b) находит минимальное из двух чисел. Иллюстрация принятия решений на основе условий.
Стрелочные функции Представлена стреловидная функция double(x), умножающая число на 2. Продемонстрирован синтаксис стрелочных функций ES6.
Колбэк Функция apply(fn, x) принимает функцию-калбэк и аргумент, вызывая их вместе. Работает концепция передачи функций в качестве аргументов.
Фильтр по длине слова Функция filterByLength(words, minLen) фильтрует массив слов, оставляя только длинные слова. Используется метод filter() для отбора элементов массива.
Работа с массивом Функция toUpperArray(arr) преобразует все строки массива в верхний регистр. Метод map() демонстрирует применение операций ко всему массиву.
Значение по умолчанию Функция power(base, exp = 2) возводит число в указанную степень, используя значение по умолчанию для показателя степени. Закрепляется понимание параметров с умолчаниями.
Проверка на чётность Функция isEven(n) проверяет, является ли число чётным. Рассматривается оператор остатка %.
Округление вниз Функция floorNumber(n) округляет число вниз, используя встроенную функцию Math.floor().
Диапазон Функция inRange(x, a, b) проверяет, лежит ли число в заданном интервале. Продемонстрировано сравнение чисел и логика ветвления.
Повтор строки Функция repeatText(text, times) многократно повторяет строку. Используется цикл for для конкатенации строк.
